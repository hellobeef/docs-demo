1. 简介

欢迎来到技术文档中心，这里提供详细的产品使用说明和开发接口文档。本网站采用简约现代设计，旨在为开发者提供清晰、高效的阅读体验。

2. 快速开始

在开始之前，请确保已经安装最新版本的开发工具：

                
npm install -g my-project-cli

2.1 初始化项目

使用以下命令创建新项目：

                
my-project create my-app
cd my-app
